CasaOS installs with a single command on any supported Linux system. Once installed, the web UI is immediately accessible from your browser — no configuration files required.

## Prerequisites

- A device running a [supported operating system](/hardware-compatibility)
- SSH access or a directly attached keyboard and monitor
- `sudo` privileges on the target machine
- An active internet connection

<Warning>
  Do not run the installer from the terminal within the CasaOS UI. It must be run from an SSH session or a directly attached terminal and keyboard.
</Warning>

## Install CasaOS

<Steps>
  <Step title="Connect to your device">
    Open a terminal and SSH into your device:

    ```bash
    ssh user@your-device-ip
    ```

    If you have a monitor and keyboard attached, open a terminal directly on the device.
  </Step>

  <Step title="Run the installer">
    Paste one of the following commands and press Enter. The installer downloads and configures all required components automatically.

    <CodeGroup>

    ```bash wget
    wget -qO- https://get.casaos.io | sudo bash
    ```

    ```bash curl
    curl -fsSL https://get.casaos.io | sudo bash
    ```

    </CodeGroup>

    The installer will display progress as it sets up CasaOS. This typically takes one to two minutes depending on your hardware and internet speed.
  </Step>

  <Step title="Open the web UI">
    Once installation completes, navigate to your device's IP address in a browser:

    ```
    http://your-device-ip
    ```

    CasaOS will display its setup screen on first load.

    <Note>
      If you don't know your device's IP address, check your router's connected devices list, or run `ip addr` on the device to find it.
    </Note>
  </Step>

  <Step title="Complete initial setup">
    The setup wizard will guide you through:

    - Creating your admin account (username and password)
    - Setting a device name
    - Reviewing any drives detected on the system

    Click **Get Started** when the wizard is complete. You'll land on the CasaOS dashboard.
  </Step>
</Steps>

## Install your first app

From the dashboard, click **App Store** in the sidebar. Browse the curated selection or search for any Docker app by name. Click **Install** on any app to deploy it with one click.

Popular apps to start with: Nextcloud, Jellyfin, Home Assistant, AdGuard Home.

## Update CasaOS

CasaOS can be updated from the UI via **Settings → Update**.

To update from the command line, run the following from an SSH session or directly attached terminal — not from the CasaOS terminal:

<CodeGroup>

```bash wget
wget -qO- https://get.casaos.io/update | sudo bash
```

```bash curl
curl -fsSL https://get.casaos.io/update | sudo bash
```

</CodeGroup>

<Note>
  Terminal-based updates must be run from an SSH session or a directly attached terminal. Running the update command from the CasaOS built-in terminal is not supported.
</Note>

## Check your version

To see which version of CasaOS is installed:

```bash
casaos -v
```
