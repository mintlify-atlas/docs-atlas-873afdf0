CasaOS is designed to be configured entirely through its web UI — no command-line configuration is required for normal use. Access settings by clicking the gear icon in the CasaOS dashboard.

## Settings available in the UI

| Setting | Description |
|---|---|
| System update | Check for and apply CasaOS updates |
| Web UI port | Change the port CasaOS listens on |
| Proxy | Configure an outbound HTTP proxy |
| SSH terminal | Browser-based SSH access to your device |
| ZeroTier | Enable remote access via ZeroTier VPN |
| User accounts | Create and manage user passwords |

## Accessing settings

Open the CasaOS web UI in your browser, then click the **Settings** (gear) icon in the top-right corner of the dashboard to access system settings.

<Tip>
  Most configuration changes take effect immediately without restarting CasaOS.
</Tip>
