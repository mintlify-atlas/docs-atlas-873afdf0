CasaOS runs on a wide range of hardware — from Raspberry Pi to Intel NUC to repurposed desktop PCs. If your device runs a supported Linux OS and uses a supported CPU architecture, CasaOS will work on it.

<Tip>
  For the most reliable experience, use a fresh install of Debian 12 on an amd64 or arm64 device. This is the configuration most thoroughly tested by the core team.
</Tip>

## Supported architectures

<CardGroup cols={3}>
  <Card title="amd64 / x86-64" icon="computer">

  </Card>
  <Card title="arm64" icon="microchip">
    64-bit ARM processors. Includes Raspberry Pi 3 and later (64-bit OS), Apple Silicon (via Linux), and most modern ARM single-board computers.
  </Card>
  <Card title="armv7" icon="microchip">
    32-bit ARMv7 processors. Includes older Raspberry Pi models running 32-bit OS and similar ARM boards.
  </Card>

## Officially supported operating systems

These operating systems are tested by the CasaOS team and are recommended for production use.

| Operating system | Status | Notes |
|---|---|---|
| Debian 12 | Tested, recommended | Best overall experience across all architectures |
| Ubuntu Server 20.04 | Tested | Supported on amd64 and arm64 |
| Raspberry Pi OS | Tested | Supports both 32-bit (armv7) and 64-bit (arm64) variants |

<Note>
  CasaOS requires a freshly installed OS for best results. Installing on a system with existing services may cause port conflicts or other issues.
</Note>

## Community-supported operating systems

These operating systems are reported to work by community members. They are not officially tested by the CasaOS team and may have edge cases or limitations.

| Operating system | Status |
|---|---|
| Elementary OS 6.1 | Tested by community |
| Armbian 22.04 | Tested by community |
| Alpine Linux | Not fully tested |
| OpenWrt | Not fully tested |
| ArchLinux | Not fully tested |

## Known compatible devices

CasaOS has been tested or reported to work on the following devices:

- **ZimaBoard** — officially supported
- **Intel NUC** — officially supported
- **Raspberry Pi** (3B, 3B+, 4B, Zero 2 W) — officially supported
- Old laptops and desktop PCs with a 64-bit CPU
- Most ARM single-board computers (SBCs) with a compatible OS

Any device capable of running a supported Linux OS on a supported architecture should work.

## What's not supported

- Windows (natively)
- macOS (natively)
- 32-bit x86 (i386) processors
- Devices with less than 1 GB of RAM (not validated)

<Note>
  Running CasaOS inside a virtual machine is possible but not officially tested. Community members have reported success with Proxmox and other hypervisors.
</Note>
