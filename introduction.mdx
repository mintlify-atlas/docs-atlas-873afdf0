CasaOS is a simple, easy-to-use personal cloud system built for home users and self-hosters. It runs on your own hardware — a Raspberry Pi, an old PC, a NUC, or a ZimaBoard — and gives you a beautiful web interface to manage files, install apps, and control your home server without needing any technical background.

## What is CasaOS?

CasaOS transforms any Linux-based computer into a personal cloud server. Instead of paying for subscription cloud services, you own your data and run your applications at home — with full control over privacy, storage, and costs.

## Get started in 60 seconds

<Steps>
  <Step title="Prepare your hardware">
    Any amd64, arm64, or armv7 device running Debian, Ubuntu, or Raspberry Pi OS will work. See [hardware compatibility](/hardware-compatibility) for the full list.
  </Step>
  <Step title="Run the installer">
    SSH into your device and run one command to install CasaOS:
    ```bash
    wget -qO- https://get.casaos.io | sudo bash
    ```
  </Step>
  <Step title="Open the web UI">
    Navigate to your device's IP address in a browser. CasaOS will guide you through initial setup.
  </Step>
  <Step title="Install your first app">
    Open the App Store, find an app like Nextcloud or Jellyfin, and click **Install**. CasaOS handles everything automatically.
  </Step>
</Steps>

## Why CasaOS?

- **No cloud required** — your data stays on your hardware
- **No technical background needed** — intuitive UI designed for home scenarios
- **100,000+ apps** — any Docker app can be installed with one click
- **Multi-architecture** — runs on amd64, arm64, and armv7 devices
- **Open source** — Apache 2.0 licensed, community-driven development
